# Test Data Files

Test minidump files used for integration testing.

## Attribution

These test files are sourced from public minidump test suites:

- `test.dmp`, `full-dump.dmp` - Windows minidump samples
- `linux-mini.dmp` - Linux minidump format sample
- `simple-crashpad.dmp` - From [Crashpad](https://chromium.googlesource.com/crashpad/crashpad/) test suite
- `pipeline-inlines-macos-segv.dmp` - From [rust-minidump](https://github.com/rust-minidump/rust-minidump) test fixtures
- `invalid-*.dmp` - Synthetic malformed dumps for error handling tests
